# 🏏 Bat Ball Stump Game

A simple and interactive cricket-themed game built using HTML, CSS, and JavaScript.

---

## 🎮 How to Play

- Choose one of the options:
  - 🏏 Bat
  - ⚾ Ball
  - 🪵 Stump
- The computer randomly selects its move
- The result is decided based on rules:
  - Bat beats Ball
  - Ball beats Stump
  - Stump beats Bat
- Your score updates after every round

---

## ✨ Features

- 🎨 Modern UI with hover animations
- 📊 Live score tracking (Wins, Losses, Draws)
- 💾 Score saved using localStorage
- 🔄 Reset button to clear score
- 📱 Clean and centered layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, transitions, gradients)
- JavaScript (DOM manipulation, localStorage)
